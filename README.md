# Nasdrovia!

Nasdrovia es un e-commerce enfocado en cervezas, creado como parte de la educación en Soy Henry en la cual se presentaba un proyecto full-stack hecho desde 0.

Las tecnologías utilizadas fueron:

//Frontend:

- HTML-CSS-JS
- React
- React - Bootstrap
- Redux

//Back End:

- Node.js
- Express
- Passport
- Sequelize

//DB:
- PostgreSQL

## How To para probar el proyecto :

Para ver la página, seguí los siguientes pasos:

- Clona el repositorio
- Crea un archivo .env en la carpeta api con el siguiente contenido:

DB_HOST:localhost<br />
DB_USER:postgres<br />
DB_PASSWORD:postgres o tu contraseña<br />
googleClientID={Google ID}<br />
googleClientSecret={Secreto de Google}<br />
facebookClientID={Facebook ID}<br />
facebookClientSecret={Secreto de Facebook}<br />

- Instala PostgreSQL en la computadora y crea una base de datos llamada "development".
- En las consolas de Facebook y Google obtene los ID de aplicación y secreto para poder utilizar los accesos en el login.
- Create una cuenta en Mailgun y obtene una API key

** Ésta aplicación no posee productos, categorias, usuarios, ni imágenes cargadas. Se deberán agregar todas desde el panel de control.
